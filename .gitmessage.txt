#   Commit Type List
#   Add      : 새로운 기능, 뷰(화면) 등 추가

#   Update   : 기능 수정

#   Style    : 코드 의미에 영향을 주지 않는 변경사항 (스타일 변경, 형식 지정, 세미콜론 누락 등)
